# Fonts Directory

This directory should contain self-hosted font files. Add the following fonts:

## SN Pro (UI Font)
- SNPro-Regular.woff2
- SNPro-Medium.woff2
- SNPro-SemiBold.woff2
- SNPro-Bold.woff2

## Newsreader (Editorial Serif)
- Newsreader-Regular.woff2
- Newsreader-Medium.woff2
- Newsreader-SemiBold.woff2
- Newsreader-Bold.woff2
- Newsreader-Italic.woff2

## MiSans (Chinese Sans)
- MiSans-Regular.woff2
- MiSans-Medium.woff2
- MiSans-SemiBold.woff2
- MiSans-Bold.woff2

## Download Sources
- **SN Pro**: Available from various font foundries
- **Newsreader**: Google Fonts (https://fonts.google.com/specimen/Newsreader)
- **MiSans**: Xiaomi official font

## Conversion
Use tools like `woff2` or online converters to generate .woff2 files from .ttf/.otf sources.
